# 🎂 Age Calculator

A simple Python script that calculates your age based on your birth year and current year. 🐍✨

## 📝 Description

This program prompts the user to enter their birth year and the current year, then calculates their age by subtracting the birth year from the current year. It prints the result to the console.

## ✅ Features

- ✔️ Takes birth year as input
- ✔️ Takes current year as input
- ✔️ Calculates age instantly
- ✔️ Works for any year, not just a fixed one
- ✔️ Simple, beginner-friendly code
- ✔️ No external dependencies required

## ⚠️ Notes

- 🗓️ Make sure to enter a valid current year greater than the birth year, otherwise the result may not make sense (e.g. negative age).
